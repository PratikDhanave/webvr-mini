    public getPixel(x: Number, z: Number, edgeFlag: Enum): Number source

    Get a Map pixel. For the diamond algorithm, this.max is a width or height. For all others, it is the length of the entire array.

    Params:

    NameTypeAttributeDescription
    x Number

    the x coordinate of the pixel (column)
    z Number

    the z coordinate of the pixel (row)
    edgeFlag Enum

    how to handle requests off the edge of the map

    • WRAP: grab from other side, divide to zero).
    • TOZERO: reduce to zero, depending on unit distance from edge.

    Return:

    Number

    the Map value at the x, z position.

    public initDiamond(w: Number, h: Number, create: Boolean, roughness: Number) source

    Create a blank heightmap in canvas ImageData format. If random === true, make a random heightmap. https://github.com/hunterloftis/playfuljs-demos/blob/gh-pages/terrain/index.html

    Params:

    NameTypeAttributeDescription
    w Number

    the width of the heightmap (x).
    h Number

    the height of the heightmap (z).
    create Boolean

    if true, make a proceedural heightmap using diamond algorithm.
    roughness Number

    if create === true, assign a roughness (0 - 1) to generated terrain.
