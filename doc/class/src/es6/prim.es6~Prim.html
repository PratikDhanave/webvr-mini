    public geometryBottomCone(the: Prim): Prim.geometry source

    type BOTTOMCONE rendered as GL_TRIANGLES. Cone structure, pointing downwards. startSlice cuts off the cone, and wraps the texture across the top. endSlize truncates the bottom of the cone, and wraps the texture across the bottom. for a cone with no caps, set startSlice and endSlize to zero. prim.dimensions = (vec4) [ x, y, z, startSlice | 0, endSlice | 0 ] prim.divisions = (vec3) [ x, y, z ]
